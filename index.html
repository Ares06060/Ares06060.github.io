观股市有感

发表于 2024-10-10 更新于 2024-10-11

股市生存法则

巴菲特说过，进入股市，你只需要遵守两条规则
rule No.1 : Never lose money
rule No.2 : Never forget rule No.1
这两句话听起来像是废话，但它蕴含着深刻的投资哲学，很难用几段话来讲明白这个核心要义，我会试着去用通俗易懂的话来阐述它。

投机

首先我想每个人进入股市都不是奔着亏钱来的吧？最近股市行情如火如荼，新老韭菜一窝蜂涌入，都想分一杯羹，就像一场在过去30多年不断重复的戏剧，最后不免一地鸡毛。
我认为这些人跟去赌场没什么区别，就赌明天大涨，如果猜对了就欢呼雀跃，期待着后天继续涨，如果跌了就伤心难过，期待后天能涨回来，我不觉得这跟赌博有什么区别。
以上行为我定义为投机，一个人如果就是抱着来股市玩一玩的想法，就像去赌场找点乐子一样，我觉得无可厚非，只要他输得起。

如果我进入股市不投机，那么我该怎么赚钱或者保证不亏钱呢？

投资

首先我们要搞明白，怎么样才算是赚钱。利润 = 卖出价格+分红-买入价格-一系列服务费和印花税，如果利润为正，则为正收益。分析这个公式我们知道，要尽可能获得更高的收益，我们要保证卖出价格高，分红收入高，与此同时保证买入价格够低，并且少交点服务费和印花税（这取决于交易平台和政策）。
对大部分人来说，追求一个比较高的卖出价格是最理想的，那么我们就来分析一下股票的价格是由什么决定的：
市值 = 市盈率（PE）*利润
其中利润是公司每年能挣多少钱，市盈率是由市场决定的。例如公司A去年净利润为1亿元，此刻市盈率为30倍，那么它当前市值就是30亿。
看到这里我们就不禁要思考应该赚哪一部分钱呢？是公司利润上涨带来的估值的提升，还是市盈率提高带来的估值提高的钱呢？
在股市中绝大多数人为了投机，希望短期内股票出现巨大的涨幅，这是由市场情绪决定的，带来的就是市盈率短期的大幅提升。
与之相反，价值投资者希望公司能够在未来持续赚取越来越多的利润，由利润驱动的市值的提升，或者带来分红，这种思路才是理性的。

总结

我们知道，每个人进入股市都是为了赚钱，或者至少保证能够跑赢通胀，为了实现这种目的我们就要知道自己应该陪伴一些优秀的企业，从他们赚的钱里分一杯羹，而不是像个赌徒一样一股脑冲进股市，追涨杀跌，热血沸腾地来，灰溜溜地去。在市场中，数以亿计的庄家，玩家以及大量的咨询使得任何人无法用任何模型可以预测股市短期的涨跌，我们能做的就是找到确定性，找到一个锚，这个锚就是公司的未来现金流折现（这个概念以后会详细讲解），当你知道你所持有的公司在未来能赚越来越多的钱，你就知道坚定的持有它，你的财富自然而然就会增长。
然而怎样找到这样的印钞机就是价值投资最难的地方，你做的就是要不断地学习，尽量找到自己看得懂的企业，然后花大量的时间去学习搞懂它。

投资就是要拥抱印钞机企业，摒弃自己不懂得垃圾公司，账户里的数字自然而然就会增长。
